FAQ

Frequently asked questions

How long does integration take?+

For most teams, full integration takes 10–30 minutes. Install the SDK, register your agents, and add one line per action. We have guides for LangChain, CrewAI, AutoGen, and vanilla Python.

Does TrustAgent add latency to my agent?+

No measurable impact. The SDK call responds in under 2ms. Your agent proceeds immediately while scoring happens in parallel.

What happens when an agent is suspended?+

The agent's status is set to "suspended" and all subsequent behavior logs return a block signal. Your code checks result.is_safe before proceeding. You receive a webhook alert and the full incident is preserved in the audit trail.

How does the ML model learn normal behavior?+

The model trains on each agent's individual history using Isolation Forest with 15 features. After 10 logs, scoring begins. After 100 logs, the baseline is mature and detection accuracy is high.

Is my agent's behavior data kept private?+

Yes. All data is isolated per organization and encrypted at rest. We never share data between organizations or use your data to train shared models.

DOCUMENTATION

Developer Documentation

Everything you need to integrate TrustAgent. Get started in under 10 minutes.

Quick Start

pip install cllova

from cllova import TrustAgent

ta = TrustAgent(api_key="your_api_key")
agent = ta.register("MyAgent")
result = agent.log("process_request", {"user": "alice"})

print(result.trust_score)   # 100.0
print(result.is_anomalous)  # False
print(result.status)        # active

Authentication

All API requests require your organization API key as the x-api-key header. Get your key from the dashboard after signing up.

SDK Methods

ta.register(name, description) — Register a new agent. Returns an Agent object with a unique ID and Ed25519 keypair.

agent.log(action, payload) — Log a behavior event. Returns trust score, anomaly flag, status, and reason.

agent.refresh() — Fetch latest agent state from the API.

agent.audit_trail() — Retrieve full behavior history for the agent.
