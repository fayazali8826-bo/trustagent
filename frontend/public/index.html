📖 Quick Start

Register an agent:

curl -X POST https://trustagent-production.up.railway.app/api/agents/register \
  -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"name":"MyAgent","description":"My AI agent"}'

Log a behavior:

curl -X POST https://trustagent-production.up.railway.app/api/agents/AGENT_ID/behavior \
  -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"action_type":"send_email","payload":{"to":"user@example.com"}}'
